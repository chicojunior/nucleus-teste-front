* SCSS
* Bootstrap
* NPM

---

##Instruções para rodar o projeto

Pré-requisitos:
* Node.js
* NPM

Baixe o projeto:

```shell
git clone https://github.com/chicojunior/nucleus-teste-front.git
```

Na raiz do projeto execute o comando:

```shell
npm install
```

Por fim execute:

```shell
npm start
```

O projeto irá executar na porta 3000.
